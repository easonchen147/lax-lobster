# 龙虾海底跑酷

基于 `Vite + TypeScript + Phaser 3` 的竖屏海底跑酷游戏原型。

## 启动

```bash
npm install
npm run dev
```

## 测试

```bash
npm run test:run
```

## 构建

```bash
npm run build
```

## 目录

- `src/scenes`：场景与页面逻辑
- `src/entities`：龙虾、障碍物、捕食者、道具实体
- `src/systems`：难度、滚动、碰撞、音效等系统
- `src/data`：游戏状态与本地存储
- `src/ui`：界面组件与视觉主题
- `src/config`：数值与配置数据
