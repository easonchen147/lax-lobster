# 游戏手感验证清单

- [ ] 点击/按住时，龙虾立即下潜，松开后立即上浮。
- [ ] PC 鼠标与移动端触控的响应延迟都在 50ms 以内。
- [ ] 0-3 秒内不会刷出伤害性障碍。
- [ ] 3-6 秒内障碍通道明显更宽，便于适应手感。
- [ ] 260 分后场景速度明显提升，空隙变窄但仍可通过。
- [ ] 380 分后会出现捕食者，且单屏最多 1 只。
- [ ] 触发护盾时龙虾不死亡，并显示同心圆护盾层数。
- [ ] 护盾达到 3 层后再次拾取，不应消耗该道具。
- [ ] 免费复活按钮可用时，点击后原地复活并获得短暂无敌。
- [ ] 免费复活耗尽后，再次死亡直接进入结算页。
- [ ] 达到 120 / 260 / 480 分等阈值时，会触发进化解锁提示。
- [ ] 进化后当前局立即应用属性变化，不会暂停游戏。
- [ ] 进入进化中心时，已解锁形态高亮，未解锁形态灰显。
- [ ] 切换已解锁形态后，重新开局会应用所选形态。
- [ ] 关闭页面重新打开后，最高分、金币、已解锁形态依然保留。
