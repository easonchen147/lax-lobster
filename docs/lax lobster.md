# 龙虾海底跑酷 完整版游戏总策划书

# 一、游戏核心概述

## 1.1 游戏定位

本游戏为**竖版无限向右滚动、PC+移动端双端互通休闲跑酷手游**，主打单手/单鼠标极简操作、轻量化竞技、养成进化与轻度商业化，核心面向全年龄段休闲玩家，兼顾手残党容错性与高手操作感，以“高风险高回报道具+龙虾进化变强+复活变现”为核心亮点，打造爽感十足、耐玩度高的海底跑酷体验，双端数据同步、操作逻辑完全统一，降低玩家上手与切换成本。

## 1.2 核心玩法

游戏全程场景**强制向右滚动、永不停止**，玩家**双端操作逻辑完全统一、无任何差异化设定**：PC端点击/按住鼠标左键操控龙虾下潜冲刺、松开左键自动上浮，移动端点击/按住触摸屏操控龙虾下潜冲刺、松开手指自动上浮；依托浮力机制，躲避珊瑚、海草、沉船等障碍，规避海鳗、石斑鱼等捕食者，拾取各类道具获取增益；通过累计分数解锁龙虾进化形态，获得属性加成，挑战更高分数，死亡后可通过看广告/消耗道具复活，追求极限高分。

## 1.3 核心特色

- **双端极致适配**：PC+移动端无缝互通，操作逻辑、响应速度、手感反馈完全统一，一套操作玩转双端，数据同步互通，适配居家PC、外出移动端多场景游玩，零切换成本。

- **手感友好**：开局专属适应期，零失败压力，极简操作快速上手

- **进化养成**：多阶龙虾进化，外观蜕变+属性增益，成长感拉满

- **风险平衡**：稀有道具刷新于危险区域，但预留安全获取路径，无必死陷阱

- **轻度变现**：广告+付费双复活模式，无逼氪、无属性碾压，公平性拉满

- **随机趣味**：海底随机小事件，丰富场景体验，避免玩法单调

# 二、游戏基础设计

## 2.1 双端统一操作细则（零差异版）

- **核心操作逻辑（双端100%一致）**：全程仅支持**单点操控**，无复合操作、无额外按键；点击/按住执行下潜冲刺，松手/松开鼠标触发自动上浮，无左右移动指令，操作判定规则、响应速度、反馈效果完全同步。

- **PC端专属适配（贴合键鼠操作）**：仅启用鼠标左键有效，右键屏蔽无响应，杜绝误触；支持短按点击（单次下潜）、长按持续下潜两种模式，判定阈值统一；鼠标悬停无操作，仅触发点击/松开动作生效，适配PC大屏点击精度，操作判定区域与移动端居中核心区对齐。

- **移动端专属适配（贴合触控操作）**：全屏触控有效，核心操作区居中放大判定范围，边缘区域判定阈值同步放宽，避免握持误触；支持单点短按、长按持续下潜，触控按压/松手判定延迟与PC端完全一致；屏蔽多指触控，仅识别单指操作，防止误操作。

- **操作响应标准（双端无偏差）**：点击/按压至龙虾动作响应延迟≤0.05秒，松手/松开鼠标至上浮动作响应延迟≤0.05秒；下潜力度、上浮速度、边界回弹等操控数值双端完全共用一套参数，无速度差、手感差。

- **场景规则统一**：画面强制向右持续滚动，仅结算/复活界面暂停，其余场景不受操作、死亡、复活影响；死亡判定标准统一：触碰障碍、被捕食者攻击、触碰屏幕上下边界，即刻判定死亡。

- **开局操作保护**：0-3秒纯手感适应期，无障碍、无敌人、强制无敌，双端可自由练习无惩罚；3-6秒仅刷新超大空隙简易障碍，操作容错拉满；6秒后正式进入正常难度，难度递增节奏双端完全同步。

- **特殊操作容错**：短时间连续点击/触控，仅识别最新一次有效指令，杜绝操作卡顿；双端均支持快速连点微调龙虾位置，连点响应速率、位移幅度完全统一，保证高手微操手感一致。

## 2.2 双端操作异常处理细则（含精准触发阈值）

本板块所有触发阈值、判定时长、计数规则**PC端与移动端完全统一**，无差异化配置，开发需同步校验双端阈值参数，确保异常处理效果一致，兼顾容错性与防误判，避免正常操作被误判为异常。

- **操作无响应异常（精准触发+处理阈值）**
**触发阈值**：玩家执行点击/按压操作后，**连续0.3秒无龙虾动作反馈**（超出正常0.05秒响应上限），判定为单次操作无响应；**10秒内累计触发3次及以上**，判定为持续性操作无响应。
        
**处理规则**：单次无响应触发0.3秒容错缓冲，重复操作一次即可激活指令；持续性无响应，弹出统一轻提示“操作失效，请重试”，提示停留2秒后自动消失，不遮挡核心画面，同时临时放大操作判定区域30%，直至恢复正常响应。
      

- **误触/多指操作异常（触发+屏蔽阈值）**
**触发阈值**：PC端点击鼠标右键/侧键、移动端检测到**双指及以上触控/滑动操作**，即刻判定为误触；移动端单次触控按压面积＞80%屏幕区域，判定为无效大面积误触。
**处理规则**：误触操作全程屏蔽，无任何动作响应、无报错提示、无游戏节奏干扰；仅保留单指单点/长按、鼠标左键有效指令，误触计数不纳入异常累计统计。
      

- **断连/卡顿操作异常（触发+缓存阈值）**
**触发阈值**：游戏帧率**连续2秒低于20帧**，或网络延迟＞3000ms且持续1秒，判定为卡顿/断连异常；**卡顿时长≤5秒**为短时卡顿，＞5秒为长时断连。
        
**处理规则**：短时卡顿自动缓存最新1条操作指令，卡顿期间龙虾维持原有上浮/下潜动作，防止突发死亡，恢复正常后立即执行缓存指令；长时断连直接暂停游戏，弹出提示“网络异常，请检查连接”，恢复联网后同步场景进度，不丢失对局数据。
      

- **边界操作异常（触发+稳控阈值）**
**触发阈值**：龙虾触碰屏幕上下边界，且**0.5秒内连续触碰2次及以上**，判定为边界操作失控异常；单次边界触碰纳入正常容错，不判定异常。
        
**处理规则**：触发异常后锁定0.4秒无敌状态+操作暂稳，强制平缓回弹，禁止短时间内连续回弹；双端边界回弹力度、无敌时长、暂稳规则完全一致，避免玩家因边界误触快速死亡。
      

- **复活/结算界面操作异常（触发+响应阈值）**
**触发阈值**：复活/结算界面中，**点击按钮区域后0.2秒无界面响应**，或点击非按钮区域，判定为界面操作异常；对局操作指令在界面开启期间，一律判定为无效操作。
        
**处理规则**：无效点击无响应，按钮点击无响应时高亮对应按钮1秒，同步提示“点击按钮确认操作”；界面关闭前，持续屏蔽对局操控指令，杜绝误触串操作。
      

- **异常兜底规则（累计触发阈值）**
**触发阈值**：同一局对局内，**操作无响应、边界失控、界面异常累计≥3次**（卡顿/断连、误触不纳入累计），触发兜底极简模式。
        
**处理规则**：自动开启极简操作模式，操作判定区域放大50%、响应阈值放宽至0.1秒，降低操作难度；双端触发条件、模式效果完全同步，对局结束后自动恢复默认操作设置，不影响后续游玩。
      

## 2.3 双端操作UI提示文案（统一优化版）

- **开局引导提示**：居中悬浮提示“点击/长按下潜，松手自动上浮”，持续3秒后消失，字体大小、颜色、位置双端适配，新手期首次进入必显。

- **操作失误提示**：触碰边界/误触时，轻量小字提示“轻点屏幕/鼠标微调位置”，不弹窗、无遮挡，1秒后自动消失。

- **异常操作提示**：操作失效/断连时，底部轻提示“操作异常，点击重试”，双端提示样式、停留时长（2秒）统一，无强制弹窗。

- **高分段操作提示**：达到1000分后，随机轻提示“长按稳下潜，连点巧避障”，辅助玩家操作，频次≤1次/局，避免干扰。

- **复活界面操作提示**：按钮下方小字提示“PC点击左键/移动端轻点对应按钮”，清晰指引双端操作，字体简洁醒目。

## 2.4 双端操作逻辑一致性管控标准

- **底层参数统一**：双端共用一套操控核心参数（响应延迟、下潜/上浮速度、判定范围），禁止单独配置差异化参数，开发需同步校验双端数值。

- **判定规则统一**：点击/长按/松手的判定阈值、有效操作区域、容错机制完全一致，PC端鼠标点击≠移动端触控加速/减速，杜绝手感偏差。

- **反馈体系统一**：操作音效、震动、画面反馈双端同步触发，延迟差≤0.03秒；长按持续下潜时，龙虾动作、拖尾特效双端无差异。

- **测试校验标准**：开发测试需双端同步操作，对比动作响应、异常处理、提示弹窗，确保无延迟、无偏差、无差异化表现，达标后方可上线。

- **异常阈值校验**：所有异常触发阈值、处理时长，双端需进行10次以上重复测试，误差≤0.05秒，确保异常判定、处理效果完全一致。

## 2.5 场景与障碍设计

### 2.5.1 场景氛围

海底纵深递进式场景，随分数升高，海水由浅海亮蓝逐步过渡至深海幽蓝，搭配漂浮气泡、游动鱼群、阳光光束、海带摇曳等装饰元素，氛围层层递进，不影响核心操作；双端自适应分辨率，PC端大屏无拉伸、移动端小屏无遮挡，核心操作区域居中。

### 2.5.2 障碍类型

- 基础障碍：珊瑚群、海草、礁石、沉船碎片，固定形态，横向排布，碰撞判定清晰

- 捕食者障碍：海鳗、石斑鱼、梭子鱼，随机突袭，移动速度快，威胁性高，预警机制明显

- 随机障碍：洋流（轻微上下推挤龙虾）、发光水母（触碰加分）、开合贝壳（彩蛋互动）

## 2.6 道具体系

### 2.6.1 道具分级与效果

- **普通道具**：护盾泡泡（抵挡一次伤害）、减速洋流（场景滚动速度小幅降低）、驱鱼剂（驱散周围捕食者），刷新概率高，分布于安全区域

- **稀有道具**：回溯珍珠（额外复活机会），刷新概率极低，分布于高危险区域，预留安全获取通道，无必死设计

### 2.6.2 道具刷新规则

单屏幕内仅刷新1个道具，避免扎堆；前期概率偏高、后期概率降低，稀有道具全局冷却，一局最多刷新1个；连续无道具触发保底机制，防止玩家全程无道具可用。
